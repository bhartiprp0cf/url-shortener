# URL Shortener

A URL shortening application built using Node.js, Express.js, MongoDB, and EJS.

## Features
- ✅ Generate short URLs
- ✅ Redirect to original URLs
- ⏳ User authentication (Coming Soon)
- ✅ Click analytics

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS

## Installation

```bash
git clone <repository-url>
cd url-shortener
npm install
npm start
